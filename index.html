엑셀 파일 준비 가이드

  1. 아래 업로드 템플릿 다운로드 버튼으로 양식을 받습니다.
  2. 시트탭 이름이 정확히 Meta_DB / Tiktok_DB / Google_DB / 키워드매칭 / Meta / Tiktok / Google 이어야 합니다. 유효 DB수는 Meta_DB/Tiktok_DB의 첫 행과 빈 행을 제외한 모든 행으로 집계합니다.
  3. 광고비는 소진됐지만 CRM 키워드 유입이 없는 소재도 광고비/클릭은 반영하고 DB수·계약수는 0으로 대시보드에 표시합니다.
  4. 키워드매칭 시트의 타입 열에 브라이언 / AI기타채널 / 신규포맷 / 유튜브 / 미분류 중 하나를 입력하면 비교표에 반영됩니다.
  5. 카빌리지 / 차맵 파일을 각각 업로드하세요.
