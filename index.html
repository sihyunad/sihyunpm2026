엑셀 파일 준비 가이드

  1. 아래 업로드 템플릿 다운로드 버튼으로 양식을 받습니다.
  2. 시트탭 이름이 정확히 Meta_DB / Tiktok_DB / Google_DB / 키워드매칭 / Meta / Tiktok / Google 이어야 합니다. 유효 DB수는 Meta_DB/Tiktok_DB의 첫 행과 빈 행을 제외한 모든 행으로 집계합니다.
  3. 키워드매칭 시트의 타입 열에 브라이언 / AI기타채널 / 신규포맷 / 유튜브 중 하나를 입력하면 비교표에 반영됩니다.
  4. 카빌리지 / 차맵 파일을 각각 업로드하세요.

구글 스프레드시트 연동 가이드

  1. 엑셀 업로드 탭의 업로드 템플릿과 동일한 시트 구조(Meta_DB / Tiktok_DB / Google_DB / 키워드매칭 / Meta / Tiktok / Google)로 구글 스프레드시트를 작성합니다. 유효 DB수는 Meta_DB/Tiktok_DB의 첫 행과 빈 행을 제외한 모든 행으로 집계하며, 키워드매칭 시트에는 타입 열을 추가할 수 있습니다.
  2. 구글 스프레드시트를 공유 → 링크가 있는 모든 사용자 (뷰어)로 설정합니다.
  3. 스프레드시트 URL을 아래에 붙여넣고 연동 적용을 클릭합니다.
  4. 데이터를 업데이트한 후에는 다시 연동 적용을 눌러 새로고침하세요.
